# Tech Tonic

## 技術棧

- Ruby 3.2.1
- Ruby on Rails 7.0.4.3
- Tailwind CSS
- PostgreSQL 14

## 快速開始

要在本地運行 Tech Tonic，請按照以下步驟操作：

1. 克隆專案到本地：

```bash
git clone https://github.com/your-username/tech-tonic.git
```

2. 設定環境：

```bash
修改 database.yml
`config/database.yml`
參照範例新增 .env
example.env
```

3. 啟動 docker-compose：

```bash
docker-compose up --build
```

4. 在瀏覽器中訪問：

```bash
http://localhost:3000/
```
